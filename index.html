Contextual Image-Based Recommendation Model For Personalized Fashion

A novel recommendation framework using Deep CNN and Seq2Seq model that captures similar purchase patterns between customers using apparel images. Pioneered the use of image contexts to create customer genomes. Recommends products with recall score of 67% and precision 70%. Tools: Python, PyTorch, Scikit-Learn, OpenCV

Report Code

Spoiler Detection Using Self-Attention Based Siamese LSTM and Convolutional Similarity

A Siamese neural network architecture with self-attention to find semantic textual similarity between movie plots and their reviews. Pioneered the application of temporal CNN to learn position-invariant features in lengthy texts. Catches spoiler reviews on IMDB with sensitivity of 78% and a specificity of 71%. Tools: Python, NLTK, Scikit-Learn, Keras/TensorFlow

Report Code

Identifying Causative Agents in Pneumonia Diagnoses Using Deep CNN

While the existing research succeeds in predicting the disease, this model identifies the type of pediatric pneumonia. This study illustrates the visual patterns in chest radiographs and how each type of pneumonia deviates from normalcy. The proposed model succeeds in identifying the causative agent with a sensitivity of 87% and a specificity of 93.25%. Tools: Python, OpenCV, Scikit-Learn, Keras/TensorFlow

Report Code

Social Media Study of Public Opinions on Abortion in the aftermath of Texas Heartbeat Act

First of its kind opinion classification model using BERT, M3 Inference and Deep Face that predicted user opinion on abortion with 96% accuracy. Based on tweets and profile information extracted from Twitter. Tools: Python, Keras/TensorFlow, Scikit-Learn, Tableau, OpenCV, NLTK

Report Code

An Enhanced Approach for Detecting Helmet on Motorcyclists Using Image Processing and Machine Learning Techniques

A 3-step method to identify motorcyclists who are not wearing helmets using traffic surveillance footage. Trained an ensemble classifier with Logistic Regression and SVM to detect absence of helmet with an accuracy of 95% Tools: Python, MATLAB, Scikit-Learn
